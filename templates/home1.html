Matricks


Welcome to Matricks!!! This is a website made by Connor Uong during the 2023-2024 academic year (freshmen year). This is my first ever solo fullstack project and I decided to make a linear algebra matrix operation solver. The reason I chose to do this project is that I wanted use coding as a way to mediate my linear algebra and programming abilities. This web application is able to solve several matrix operations such as, inverse, determinant transpose, and more!!! Through this project, I was able to reinforce several of my programming skills as well as a way to learn new programming technologies.
